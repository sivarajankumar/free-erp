Configuration de freeERP

Catégories

Les catégories sont utilisées pour les classement de données.
Selon les items la recherche s'en retrouve accélérée.

Etat/Type

L'état d'un élément est sont statut transitoire ou permanent.
Le type est utilisé pour définir une typologie (par exemple pour les tiers on peut définir des tiers de type client ou fournisseur.

Donnée de référence

Les données de référence sont la plupart du temps associé à une norme ou une habiture (ex : titre peut être M., Me. ou Mlle).
